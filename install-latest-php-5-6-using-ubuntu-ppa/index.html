Install latest PHP 5.6 using Ubuntu PPA

Nur Rony

Polyglot Programmer. Blindly love JS. An automation freak who love to cook code in React, Redux, RxJS and serve them using Docker. Married to the most beautiful girl.

It is very easy installing latest PHP 5.6 using PPA

sudo add-apt-repository ppa:ondrej/php5-5.6 -y  
sudo apt-get update  
sudo sudo apt-get install make php5\  
php5-mongo \  
php-pear \  
php5-dev \  
php5-mysql \  
php5-common \  
php5-cli \  
php5-curl \  
php5-fpm \  
php5-mcrypt \  
php5-xmlrpc \  
php5-gd \  
php5-json \  
php5-readline \  
mcrypt \  
curl \  
dh-make \  
libmcrypt-dev \  
mysql-server-5.6 \  
mysql-client-5.6 \  
php5-sqlite \  
php5-oauth

You can remove or add extensions that is not needed by you. You may need to install software-properties-common previously python-software-properties
