Installing Java/JDK version on Ubuntu
19 Jan 2016
Installing Java on Ubuntu is a piece of cake. Run the following commands in your terminal sudo add-apt-repository ppa:webupd8team/java -y sudo apt-get update echo oracle-java8-installer shared/accepted-oracle-license-v1-1 select true | sudo /usr/bin/debconf-set-selections sudo apt-get install oracle-java8-installer Now Ubuntu package manager will do the rest for you.
