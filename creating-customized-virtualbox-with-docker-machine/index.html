Creating Customized Virtualbox with Docker Machine

22 Mar 2016

It is very easy to customize the configuration of Virtualbox using Docker Machine. To create a Virtualbox run the following command from your command line.

Creating Virtualbox

docker-machine create --driver virtualbox <virtualbox-name>

Your virtualbox will be created on the fly with its default configuration. But there is a problem. It will take all of your CPU cores and sometimes the memory is too low to work with.

Virtualbox with Custom Configuration

So you want to create Virtualbox specifying your desired configuration. To create a Virtualbox with custom configuration run the following command 

docker-machine create --driver virtualbox --virtualbox-cpu-count <cpu-number> --virtualbox-memory "<mem-in-MB>" --virtualbox-disk-size "<size-in-MB>" <virtualbox-name>

Please replace <blah> with your desired values. Gotta go. If you have any question shoot it out. I am all ears
