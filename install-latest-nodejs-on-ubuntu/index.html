Install latest NodeJS on Ubuntu

25 Jan 2016

I was looking for PPA to install latest NodeJS but found no PPA to install yet. So I install it using NVM as follows

Installing and configuring NVM

Installing NVM is a piece of cake. You need to execute the following command

With cURL: 

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash

with wget 

wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.30.2/install.sh | bash

After installation please add these lines to your ~/.bashrc, ~/.profile, or ~/.zshrc file to have it automatically sourced upon login: 

export NVM_DIR="$HOME/.nvm"  
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh"

Find and Install latest NodeJS Version

Run the command in your terminal 

nurrony@devmachine: ~$nvm ls-remote

Then install your desired NodeJS version executing the following command 

nvm install <your-nodejs-version>
