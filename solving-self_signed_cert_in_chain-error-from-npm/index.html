Solving 'SELF_SIGNED_CERT_IN_CHAIN' Error From NPM

06 Apr 2016

I encountered this weird problem couple of years ago. Same problem is occurred for one of my friend while using node 0.12.7 with npm 2.11.3. He almost gone mad. This happened because npm no longer supports its self-signed certificates. So when you are using older version of NodeJS along with NPM try the following commands to solve this creepy error

Solutions

Upgrade your npm version 

npm install npm -g --ca=null

or

Tell your current version of npm to use known registrars 

npm config set ca=""

if all else fails, upgrade node.js ;-) :-p
